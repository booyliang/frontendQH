## Props

| 名称			| 类型		| 说明							| 默认值			|
| ---			| ---		| ---							| ---			|
| `fullscreen`	| `Boolean`	| 全屏的。						| `false`		|
| `icon`		| `String`	| 图标 URL （**非 `class`**）。	| -				|
| `title`		| `String`	| 标题。							| -				|

## Slots

| 名称		| 说明						|
| ---		| ---						|
| `default`	| 任何内容。将追加到标题下方。	|
