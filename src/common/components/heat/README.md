## Props

| 名称		| 类型				| 说明										|
| ---		| ---				| ---										|
| *`data`	| -					| 参见 [`FlowItem`](../flow-item)。			|
| *`heat`	| `Array<String>`	| 热度字段的名称，参见[预设字段](#预设字段)。	|

## 预设字段

热度字段的可选值。

| 名称			| 含义		|
| ---			| ---		|
| `'answer'`	|	回答。	|
| `'comment'`	|	评论。	|
| `'discuss'`	|	讨论。	|
| `'favorite'`	|	收藏。	|
| `'like'`		|	点赞。	|
| `'view'`		|	浏览。	|
