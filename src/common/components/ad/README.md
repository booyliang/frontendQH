## Props

| 名称			| 类型							| 说明															| 取值										| 默认值		|
| ---			| ---							| ---															| ---										| ---		|
| `at`			| `String` / `Element` / `Vue`	| 期望插入本广告的目标位置。会被转换为 DOM 元素，广告将插入到该元素后。	| 类型为字符串时，值为 CSS 选择器。				| -			|
| *`keyword`	| `String` / `Array`			| 关键字。														| -											| -			|
| *`type`		| `Number`						| 类型。															| `1`：轮播。<br>`2`：图文。<br>`3`：大图文。	| -			|
