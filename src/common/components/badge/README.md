### Badge 标记

#### 示例

[Example](http://localhost/demo/badge)

#### Props

| 参数          | 说明            						| 类型    | 可选值    | 默认值         | 是否必填|
|---------------| -------------------------------------- | ------- | --------------- | ------------- |------|
| value | 显示值 | String, Number | —— | —— | 是 |
| max | 最大值，超过最大值会显示`${max}+`,value必须是number类型 | Number | —— | —— | 否 |
| hidden | 隐藏 badge | Boolean | true/false  | false | 否 |
