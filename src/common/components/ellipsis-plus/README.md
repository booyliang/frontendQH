### ellipsis-plus

### 参数说明
| 名称			| 类型		| 说明						 |
| text			| String 	|待显示的文本（可以是图文，数字，英文）|
| line			| String	|显示的行数，默认为2				|
| showButton	| Boolean	|是否显示组件内置按钮。默认为true	|
| expandText	| String	|展开按钮的文案					|
| collapseText	| String	|收起按钮的文案					|
| ellipsis		| String	|自定义省略符号，默认为...		|
