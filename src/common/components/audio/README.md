# 组件说明文档

## Props

| 参数          | 说明            						| 类型    | 可选值    | 默认值         | 是否必填|
|---------------| -------------------------------------- | ------- | --------------- | ------------- |------|
|   src			|	音频资源地址							|	String	|	-	|	-	|	是	|
|   time		|	音频时长,毫秒数						|	String,Number	|	-	|	-	|	是	|
|   play		|	是否可播放						|	Boolean	|	true/false	|	true	|	否	|
